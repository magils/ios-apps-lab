## FlagGuess App

This is an iOS app inspired on "Guess the Flag" game from Hacking with Swift (https://www.hackingwithswift.com/read/2/overview).

This version is a bit more advance and refined. This include:

- Multiple option selection.
- Automatic banner at the top when answer is correct or not.
- Reset feature.
- Graphically more pleasant.

The flags pictures come from the the Hacking With Swift Excercises.

This app was tested successfully on the following physical devices:

- iPhone SE (1st Gen).
- iPhone SE (2nd Gen).
- iPhone 7 Plus.
- iPhone XR
- iPhone 16 Pro Max.
